# Dashboard Snippets Library

This folder contains reusable React snippet patterns tailored to your current dashboard architecture.

## Structure

- `snippets/services/` — API client and service hook examples
- `snippets/routes/` — auth and role-based route guards
- `snippets/forms/` — form layout, input field, and page sample
- `snippets/tables/` — table skeletons and reusable row/table components
- `snippets/charts/` — Apex chart wrapper and card pattern
- `snippets/modals/` — animated modal shell and confirmation modal
- `snippets/loading/` — page-level and inline loading state patterns

## How to use

1. Copy a snippet file into `src/` when you want a reusable implementation.
2. Adapt imports to match your actual folder structure.
3. Keep the patterns consistent by reusing the `snippets/*` layout as your team grows.
